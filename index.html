AI API Reverse Proxy

当前使用域名：

${origin}

OpenAI

Base URL:
${origin}/openai

官方 SDK 示例：
client = OpenAI({
  baseURL: "${origin}/openai",
  apiKey: "YOUR_API_KEY"
})

x-api-key: YOUR_API_KEY

Authorization: Bearer YOUR_API_KEY

Gemini
